Brimstone Developer Documentation

[This is preliminary documentation and is subject to change.]

Brimstone is a high-performance Hearthstone simulator written in C#, designed to make it simple and fast for developers to experiment with AI, Machine Learning, game formats, card balance and new cards and mechanics.

 Caution
Brimstone is currnetly in pre-alpha and is not a finished product. Significant aspects of gameplay are missing or incomplete, and there may be bugs. Please help contribute (see the [TODO: Contributing section]) if you'd like to help out!
Requirements

Brimstone is written in C# and supplied as a Visual Studio solution (.sln).

You must have a compiler which supports C# 6 or later in order to build Brimstone. Brimstone targets .NET Framework 4.6.1 and is compatible with Visual Studio 2015 and Mono 4.4 or later.

Installation

To obtain Brimstone:

  • If you intend to use Brimstone in your experiments or research project as an API user only, we recommend downloading the latest Brimstone release from GitHub. This is the easiest way to obtain the software.

  • If you would like to contribute towards Brimstone's development, we recommend that you fork the latest build of Brimstone on GitHub so that you can make pull requests.

  • Brimstone receives frequent updates between releases. If you want the absolute latest build, clone the latest build of Brimstone on GitHub. Note that these builds may contain breaking bugs.

 Important

Brimstone uses git submodules. To ensure you clone the complete codebase, ensure you clone using the command: git clone --recursive https://github.com/HearthSim/Brimstone or tick the Recursively Clone Submodules box in Visual Studio's Team Explorer when cloning.

Building with Visual Studio

Load Brimstone.sln and choose Build Solution from the Build menu

Building for Mono at the command line

Type xbuild /p:Configuration=Release Brimstone.sln

Note that Brimstone is merely a black box API providing simulation capabilities and does nothing on its own. [TODO: Link to tutorials]
