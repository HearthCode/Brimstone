Pre-Alpha Update 2 was completed on August 23rd 2016

Changes in This Release

  • ActionQueue history

  • ActionQueue in-place action replacement with ReplaceAction<T>

  • Optimization settings control

  • Replace EntityGroup with mass cloning API

  • Parallel cloning

  • Use a Deque instead of a Queue for ActionQueue and enable front insertions

  • Ability to send user-defined data to ActionQueue actions

  • Fuzzy game state equivalence testing

  • Fuzzy game state equivalence comparer implementations

  • Game trees and game nodes

  • Probabilistic (random) outcome tree search API

  • Parallel tree searching

  • Asynchronous API

  • Pretty game state output

  • Benchmarking tool

  • BrimtoneVisualizer is now scriptable
